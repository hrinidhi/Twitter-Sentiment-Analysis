# Twitter Sentiment Analysis

## Overview
This project uses Natural Language Processing (NLP) and Machine Learning to classify tweets as positive or negative.

## Technologies Used
- Python
- Pandas
- NLTK
- Scikit-learn
- Matplotlib

## Model
- TF-IDF Vectorizer
- Logistic Regression

## Results
- Accuracy: 95.03%

## Dataset
Twitter Sentiment Dataset
